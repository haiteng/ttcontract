# TT合约
整合了一元购一元夺宝+事件合约玩法的小程序，TG小程序完整版本，包含前后端，后台管理
# 架构
后端Java17开发，RuoYi-Vue框架，springboot3
前端vue3开发，RuoYi-App框架，基于uniapp+uniui封装的一套基础模版，进行开发TG小程序版本
Redis进行延时队列开奖
mysql存储数据
# 功能描述
一元购模式，含有自动下单机器人，指定开奖人设置；交易使用USDT-TRC20充值，系统支持自动检测充值是否成功。
中奖TG群通知，TG小程序通知均有。分享链接，二级抽佣，抽佣比例均可以设置
# 成品部署或定制化开发
本产品支持二次定制开发，也可以直接部署赚钱，目前TT合约运营了半年了，代码已经很完善了，部署即可赚钱；目前
